Aire Acondicionado: Información Básica

¿Qué es un aire acondicionado?

El aire acondicionado es un sistema que permite regular la temperatura, humedad y calidad del aire en espacios interiores. Su objetivo principal es proporcionar confort térmico.

Tipos de aire acondicionado

  • Split: Unidad interior y exterior, ideal para hogares.
  • Portátil: Fácil de mover, no requiere instalación fija.
  • Central: Sistema más complejo para climatizar grandes espacios.

Consejos de uso eficiente

  1. Usar el aire a 24°C para equilibrar confort y consumo.
  2. Mantener puertas y ventanas cerradas.
  3. Limpiar filtros regularmente.
